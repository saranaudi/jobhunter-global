✅ The Short Version

  • Your profile data (name, email, phone, skills) is saved only on your own device — never on our servers.
  • Your Anthropic API key is stored locally only and is never transmitted to us.
  • The only external network call is to Anthropic's API, initiated only when you click "Tailor Resume".
  • We do not collect analytics, use tracking pixels, serve ads, or sell any data.
  • We do not read content from the job sites you visit — we only write to form fields when you click Autofill.
  • You can delete all your data at any time by clearing the extension's storage in Chrome settings.

1b. Data sent to Anthropic when you tailor a resume

When you click "Tailor Resume as Hiring Manager", the extension makes a single HTTPS POST request to api.anthropic.com/v1/messages. This request contains:

  • Your profile data (name, role, experience, skills, summary, work history) — or the text extracted from your uploaded resume file
  • The job description you have pasted or loaded
  • Your Anthropic API key (sent in an HTTP header directly to Anthropic — never to our servers)

Important: This request goes directly from your browser to Anthropic's servers. JobHunter Global has no server of its own and never receives, intercepts, or stores this data. Anthropic's privacy policy governs how they handle API requests: anthropic.com/privacy.

1c. Uploaded resume file

If you upload a resume file (PDF, DOCX, or TXT), it is read by the extension in your browser using the standard FileReader API. The extracted text is held in memory only for the duration of the AI tailoring request. It is not saved to storage and is not transmitted anywhere except as part of the Anthropic API call described above.

2. How We Use Your Data

Your data is used solely to power the features you explicitly activate:

  • Autofill: Profile fields (name, email, phone, location, skills, experience, LinkedIn, GitHub) are written into job application form inputs on supported job sites — only when you click the Autofill button or the floating ⚡ button on a page.
  • AI Resume Tailoring: Your profile data and the job description are sent to the Anthropic API to generate a tailored resume — only when you click "Tailor Resume as Hiring Manager".
  • Job Search: The role, location, and filters you enter are used to construct search URLs that are opened in new browser tabs. This data is not stored.
  • Interview Prep: The role title extracted from the AI response is used locally to generate resource links. No external call is made.

We do not use your data for advertising, profiling, research, or any purpose beyond the features described above.

3. Where Data Is Stored

All profile data is stored using the Chrome Extension Storage API (chrome.storage.local). This means:

  • Data is stored on your local device, inside Chrome's sandboxed extension storage.
  • It is not accessible by websites, other extensions, or other applications on your device.
  • It is not synced to Chrome Sync or Google's servers unless you have explicitly enabled extension storage sync (which this extension does not use — we use storage.local, not storage.sync).
  • Data persists until you uninstall the extension or manually clear it.

How to delete your data

To permanently delete all data stored by JobHunter Global:

  1. Open Chrome → Settings → Privacy and Security → Site Settings
  2. Click "View permissions and data stored across sites"
  3. Search for "JobHunter" and click "Clear data"

Alternatively, uninstalling the extension removes all locally stored data.

5. Chrome Permission Justifications

The following permissions are declared in the extension manifest. Each is used exclusively for the purpose stated:

PermissionWhy it is needed
activeTab To detect which job site you are currently on (used to show autofill status in the popup) and to inject the autofill script into the active tab when you click the Autofill button. Only the current tab is accessed, only when you initiate an action.
scripting To inject content.js into job site pages when you click Autofill. The script writes your profile data to visible form fields. It does not read page content, scrape data, or execute any remote code.
storage To save your profile data (name, email, phone, skills, API key, etc.) locally on your device using chrome.storage.local. No data is sent to any server.
tabs To open job site search result pages in new browser tabs when you click "Apply" or "Search Jobs". Also used to read the active tab's URL to detect which job site you are on. Tab content is not read.
contextMenus To add a right-click "Autofill this form" menu item, giving you a faster way to trigger autofill without opening the popup when you are filling a long application form.
Host permissions (40+ job sites) Required to allow content.js to run on supported job sites (Naukri, LinkedIn, Indeed, etc.) so it can write to form fields. The script runs only on these specific domains and only writes data — it does not read or transmit page content.
api.anthropic.com Required to make the HTTPS API call to Anthropic's Messages API when you initiate AI resume tailoring. This is the sole external network call the extension makes.

6. Data We Never Collect or Sell

We explicitly confirm the following, consistent with Chrome Web Store developer programme policies:

  • We do not sell or transfer user data to third parties.
  • We do not use user data for purposes unrelated to the extension's single purpose (job search assistance).
  • We do not use or transfer user data to determine creditworthiness or for lending purposes.
  • We do not use your data to train AI models. The Anthropic API processes your request statelessly — no data is retained by Anthropic beyond their standard API processing terms.
  • We do not monetise your data in any form. Our revenue model is a one-time Pro upgrade ($4.99) for additional AI builds — not data.

7. Children's Privacy

JobHunter Global is designed for job seekers and working professionals. It is not directed at children under the age of 13 (or the applicable minimum age in your jurisdiction). We do not knowingly collect personal information from children. If you believe a child has provided personal data through this extension, please contact us immediately and we will take steps to remove it.

8. Your Rights & Controls

Because all your data is stored locally on your device, you have full control at all times:

  • Access: Open the Profile tab in the extension to view all stored data.
  • Edit: Update any field in the Profile tab and click Save.
  • Delete: Clear individual fields in the Profile tab, or clear all extension data via Chrome Settings → Privacy → Site Data.
  • Portability: Your data is standard text — you can copy it from the Profile tab at any time.
  • Withdraw consent: Stop using the extension and uninstall it. All local data is deleted on uninstall.

If you are located in the European Economic Area (EEA), United Kingdom, or other jurisdictions with data protection laws (including India's DPDP Act 2023), you may have additional rights such as the right to erasure and the right to object to processing. Since we do not hold your data on any server, erasure is achieved by clearing local storage or uninstalling the extension.
